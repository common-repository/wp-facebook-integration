== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Facebook Integration Menu Link or Settings link on plugins page to use the plugin.
4. Add your Facebook Page ID and use the shortcode on any of your pages to use your Facebook Gallery Photo Album.

== Frequently Asked Questions ==

= How to get access token to my Facebook page? =
Follow the instructions in the settings page - general tab (create app, get token for your page, make it non expiry, paste it in the token field)

= Can I configure the Number of Pictures displayed in the Gallery? =
Yes, you define it from Settings Page.

= Can I get a slideshow from my Facebook images ? =
Yes, but only available on the PRO version.

= Can I get a list of my Facebook events on my fan page? =
Yes, but only available on the PRO version.
